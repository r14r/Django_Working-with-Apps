Help & Documentation

Welcome to Django App. Here is a quick overview of the available features.

Registration & Login

  • Create a new account via Register.
  • After registering, confirm your email address with the link sent to your inbox.
  • Use the Login page to sign in with your credentials.
  • Forgot your password? Use the Forgot password? link on the login page.

Todos

  • Create, edit, and delete to-do items from the Todos page.
  • Toggle an item's status between Open and Done using the check button.

Profile

  • View your account information on the Profile page.
  • Change your password via the Change Password button on the profile page.

Administration Staff only

  • Manage users and assign roles via User Management.
  • Create and edit permission groups via Group Management.
